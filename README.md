## Running Astrofox

### Get the source

```
git clone https://github.com/astrofox-io/astrofox.git
```

### Install dependencies

```
yarn install
```

### Install FFmpeg

```
yarn install-ffmpeg
```

### Start development server

```
yarn dev
```

### Start Electron

In a separate console window:
```
yarn start
```
